To enable email alerts, add these environment variables to your .env file:

# SMTP Server Settings
SMTP_HOST=smtp.gmail.com
SMTP_PORT=587
SMTP_SECURE=false
SMTP_USER=your-email@gmail.com
SMTP_PASS=your-app-password

# Email Addresses
SMTP_FROM=Server Monitor <your-email@gmail.com>
ALERT_EMAIL_TO=alerts@yourdomain.com

# Multiple recipients (comma-separated)
# ALERT_EMAIL_TO=admin@example.com,ops@example.com,alerts@example.com

üìù Common SMTP Providers:

  • Gmail: smtp.gmail.com:587 (use app password, not regular password)
  • Outlook/Office365: smtp.office365.com:587
  • SendGrid: smtp.sendgrid.net:587
  • Mailgun: smtp.mailgun.org:587

After updating your .env file, restart the container with:
docker compose restart server-monitor
